# Binance API
 
## Run the Jupyternotebooks in `analysis/` in this order:
1. `get_deposit_withdrawal_history.ipynb` : fetches user account deposit and withdrawal history
2. `get_trade_history.ipynb`: fetches user trade history
3. `get_conversion_rate.ipynb`: estimates the BTC and USDT value of each trade using Binance trade history
4. `get_average_price.ipynb`: calculates average buy/sell price of each coin in BTC and USDT
